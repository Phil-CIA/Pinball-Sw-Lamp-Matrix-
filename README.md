# Pinball Lamp Matrix

## Project Overview

This project is focused on implementing an 8x4 lamp matrix which utilizes VNQ7E100AJTR high-side row drivers, comparator switch inputs, and 74HC595 shift registers operating at 3.3V.

## Current Architecture
- **Lamp Matrix**: 8 rows and 4 columns
- **Drivers**: VNQ7E100AJTR for row driving
- **Input**: Comparator switches for user input
- **Shift Register**: 74HC595 to manage lamp states
